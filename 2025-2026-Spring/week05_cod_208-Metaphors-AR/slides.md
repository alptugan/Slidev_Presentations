# Steps of Metaphor Creation 

<div v-click>

**Step 1**: Determine the qualities of the subject that you want to express via metaphors.

</div>

<div v-click>

**Step 2**: Create a list of objects that share some common features with the thing to be described with a metaphor.

</div>

<div v-click>

**Step 3:** Compare the feature sets of the candidate objects (Wheel) with the feature set of the metaphoree (Leg). Features that match and those that don't match should be noted during this comparison. 

</div>

<div v-click>

**Step 4:** Choose the common ones and eliminate the non-mutual points. It doesn’t matter how extensive the list is.
</div>

<div v-click>

**Step 5:** You can either eliminate the <i>mismatching</i> or <i>negating</i> points to construct the metaphoric relation between objects.
</div>
